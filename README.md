# 🎓 NjeHub

Modern egyetemi rendezvényszervező platform Android Studio és Java alapokon.

---

## ✨ Funkciók

### 📅 Eseménykezelés
- Egyetemi események létrehozása és kezelése
- Esemény részletező oldal
- Esemény törlés
- Valós idejű keresés
- Esemény statisztikák

### 👥 Résztvevőkezelés
- Résztvevők hozzáadása eseményekhez
- QR-kód generálás
- Szerepkör kezelés
- Eseményhez kapcsolt résztvevők

### 🧩 Konferencia szekciók
- Konferencia szekciók létrehozása
- Előadó kezelés
- Terem és időpont kezelés
- Eseményhez kapcsolt szekciók

### 🗺️ Campus információk
- Zoomolható campus térkép
- Esemény információs rendszer
- Hasznos tudnivalók kezelése

### 🎨 Modern felhasználói felület
- Animált splash screen
- Smooth animációk
- RecyclerView animációk
- Glow gomb effektek
- Modern dark mode design
